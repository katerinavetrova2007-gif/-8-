Лабораторная работа по геометрическим фигурам

Описание проекта
Проект демонстрирует создание статических и динамических библиотек на C++ для вычисления площади и периметра геометрических фигур.

Структура проекта
- `GeometryLib.h/.cpp` - статическая библиотека
- `GeometryDll.h/.cpp` - динамическая библиотека
- `TestGeometryLib.cpp` - тесты для статической библиотеки
- `TestGeometryDll.cpp` - тесты для динамической библиотеки
- `Main.cpp` - основная программа

 Сборка проекта

### Visual Studio 2022

#### Для статической библиотеки:
1. Создайте новый проект "Статическая библиотека"
2. Добавьте файлы `GeometryLib.h` и `GeometryLib.cpp`
3. В свойствах проекта убедитесь, что выбран тип конфигурации "Static Library (.lib)"

#### Для динамической библиотеки:
1. Создайте новый проект "Библиотека DLL"
2. Добавьте файлы `GeometryDll.h` и `GeometryDll.cpp`
3. В свойствах проекта добавьте макрос препроцессора `GEOMETRYDLL_EXPORTS`

#### Для тестовых программ:
1. Создайте проект "Консольное приложение"
2. Добавьте соответствующий тестовый файл
3. В свойствах проекта добавьте ссылки на созданные библиотеки
